Example: Simple NPV calculation (Python)


def npv(rate, cashflows):
    """
    rate: discount rate (e.g., 0.1 for 10%)
    cashflows: iterable of cashflows where cashflows[0] is t=0
    """
    return sum(cf / (1 + rate) ** t for t, cf in enumerate(cashflows))

# Example usage
rate = 0.08
cashflows = [-1000, 300, 400, 500]
print("NPV:", npv(rate, cashflows))

Financial Formulae

Net Present Value (NPV):

\( \mathrm{NPV} = \sum_{t=0}^{T} \dfrac{C_t}{(1+r)^t} \)

Internal Rate of Return (IRR) definition (root of NPV):

Find \( r \) such that \( \sum_{t=0}^{T} \dfrac{C_t}{(1+r)^t} = 0 \).

Speaker note: Mention that NPV uses discount rate r; IRR is the rate making NPV zero. Use these when comparing projects.
